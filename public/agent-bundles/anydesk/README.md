# AnyDesk Bundle Directory

Drop the AnyDesk installer files here for each platform:

- `AnyDesk.exe` — Windows installer
- `anydesk.rpm` — RHEL/CentOS package
- `anydesk.deb` — Debian/Ubuntu package (if needed)

These files are bundled with the Vanguard agent installer when the "Bundle AnyDesk" option is enabled.
